# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

We are committed to providing a welcoming and inclusive experience for everyone.
Please be respectful, constructive, and kind.

## Reporting

If you experience or witness unacceptable behavior, please open an issue or
contact the maintainers directly.
